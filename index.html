YouTube Video Converter MP3 Downloader

Esta é uma aplicação para baixar vídeos do YouTube como arquivos de áudio MP3. A interface permite que você insira a URL do vídeo do YouTube, acompanhe o progresso do download e veja uma lista dos downloads concluídos.

Funcionalidades

  • Baixar vídeos do YouTube como arquivos MP3.
  • Exibir o progresso do download em tempo real.
  • Exibir uma lista de downloads concluídos.
  • Limpar a URL inserida.
  • Notificações de conclusão de download.
